# 接口测试平台后台
  功能还不完善，可能会有错误，后面有时间会更新
# 基本功能 
* 项目管理
* 测试环境管理
* 测试用例管理
* 测试集管理
* Jenkins集成

# 技术栈

  >  后台

   * SpringBoot
   * Mybatis
   * redis
   * rest-assured
  
> 前端（[前端地址请点击](https://github.com/uestctang/Daisly_Front.git "接口测试平台前端地址")）
   
   * vue2
   * vuex
   * vue-router
   * axios
   * iview
